# Custom Project Rules

- **Mobile-Only Changes**: All future layout, styling, and responsiveness changes must target mobile view only (max-width: 768px). Do NOT make any changes to desktop layout or desktop styling unless the user explicitly asks for a desktop change.
- **Desktop View Freeze**: The desktop view layout, desktop styling, and desktop responsiveness are frozen. No changes to desktop view are allowed under any circumstances unless explicitly requested by the user.
- **Rings Page Only**: All future changes to the website must be made exclusively to the Rings page (`malana-collection/malana-collection.html` and its associated CSS files `malana-collection.b1c76433.css`, `malana-collection.74605c05.css`). Do NOT make changes to any other page, file, or asset on the website unless the user explicitly asks to work on a different page.
